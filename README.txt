# Code Quality Analyzer

## 📌 Overview
This project is a **Flask-based Code Quality Analyzer** that evaluates Python scripts using **Pylint** and generates:
- **Code quality reports** with detailed analysis.
- **Dependency graphs** showing module relationships.
- A **web interface** for uploading and analyzing Python files.
